# Hyderabad Heritage AI Guide

A RAG-based AI chatbot for Hyderabad tourism.

## Features

- RAG Retrieval
- FAISS Vector Search
- Groq LLM
- Flask Web UI
- Hyderabad Heritage Guide
- Tourist Recommendations

## Tech Stack

- Python
- Flask
- FAISS
- Sentence Transformers
- Groq
- HTML/CSS

## Installation

pip install -r requirements.txt

## Run

python app.py
